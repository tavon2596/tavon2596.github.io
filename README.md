# Personal Website Portfolio

This repository contains a professional one-page portfolio website for Tavon Gordon. The site includes sections for About, Previous Work, Projects, and Contact, as well as custom CSS and JavaScript interactions.

## Files
- `index.html` – Main website page.
- `css/styles.css` – Global site styling and theme customization.
- `js/script.js` – JavaScript for contact form validation and dark mode toggling.
- `Tavon_Gordon_Resume_35202643304.pdf` – Resume linked from the Previous Work section.
- `journal.md` – Project journal documenting development progress and sources.

## How to Run
1. Open `index.html` directly in a browser.
2. Or, host the repository with GitHub Pages:
   - Push this repository to GitHub.
   - In GitHub, go to `Settings` → `Pages`.
   - Select the `main` branch as the source and click `Save`.
   - The site will deploy at `https://<username>.github.io/`.

## Features
- Navbar with one-page navigation.
- About section with professional introduction.
- Previous Work section with resume download and skills accordion.
- Projects section with a Bootstrap carousel and project cards.
- Contact form with JavaScript validation and user feedback.
- Dark mode toggle implemented with custom JavaScript and CSS.

## Libraries and Frameworks Used
- Bootstrap 5 (CSS and JS) via CDN.
- Vanilla JavaScript for interactive behavior.
- Custom CSS for consistent design across components.

## Notes
- The project is designed as a single-page website.
- The contact form is interactive and validates input before submission.
- The site includes Bootstrap components beyond standard class usage, such as accordion and carousel.
